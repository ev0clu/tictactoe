# **Tic Tac Toe**

Tic Tac Toe game has built. You should put 3 'X's or 'O's in a row to win.

### Demo: [Link](https://ev0clu.github.io/tictactoe/)

## Features

-   Player vs Player
-   Restart game option
-   3 rounds need to win
